Welcome to How to Blog!

This is a brief guide on how to blog (Edition 2017). If you are new to blogging and want to get started just follow the simple steps and write your first blog.

Decide what you want to blog about

  • Blog about something with plenty of room for discussion. A blog requires a lot of content to get going and to remain interesting.
  • Blog about something you enjoy. If you don't care about your topic, writing about it will bt drudgery.
  • Choose a niche in which you can establish yourself as an authority. You will probably not be the first person to blog about the topic you choose.

Choose a blogging platform

  • Choose a platform that is flexible, easy to use and shared by a community of people. There a lot of platform to choose from with a lot of variety in functionality. There are mainly two types of platforms.
  • Hosted - With this type of platform, you don't have to worry about setting up and managing your server. But on the downside, you don't get to choose between a lot of themes and templates.
  • Self-hosted - With this type of platform, you have a lot of themes to choose from and you have code level access. But again on the downside, you have to manage you server yourself only.

Finding a host and a domain name

  • Choosing a proper host is important. A host provides server space for your blog or site. Choose a popular host so that more people can read your blog.
  • Pick a relatable domain name for your blog. A domain name is a web address, for example, JaskaranBlogs.com. Choose a domain name that is easy to remember, spell and share.
  • Sometimes, your host may require you to install WordPress. In this case, use the link provided by the server to install it or you may also download from the link provided here. Install WordPress

Your blog is ready to use

  • Congratulations! You have a completed setting up your own blog. Now, design your blog and start sharing information, media, etc. If you are required to enter your WordPress details, check your email that you provided while installing WordPress.
